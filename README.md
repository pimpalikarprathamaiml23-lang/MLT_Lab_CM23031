# 📖 Course Overview

This repository contains all practical assignments and experiments completed for the **Machine Learning Tools Lab** course.

The objective of this lab is to understand the **fundamentals of Machine Learning** and apply them using modern tools such as Python and ML libraries.

The lab focuses on:

* Understanding machine learning fundamentals
* Implementing ML algorithms
* Working with datasets and synthetic data
* Data visualization and analysis
* Building predictive models
